How to use this on LinkedIn Featured

LinkedIn Featured does not host raw HTML files directly.

Best options:
1. Upload this folder to GitHub Pages, Netlify, Cloudflare Pages, or your own website.
2. Copy the public URL.
3. LinkedIn Profile > Featured > + > Add a link.
4. Paste the URL.
5. Use title: Application Security & DevSecOps Portfolio

Alternative:
Take a screenshot of the page and upload it as Featured media, but the live link is better.
